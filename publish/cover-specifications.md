# Cover Design Specifications

## Technical Requirements for All Platforms

### **Image Specifications**

#### Apple Books (iTunes)
- **Format**: PNG or JPEG (sRGB color space)
- **Minimum Size**: 1400 x 1400 pixels
- **Recommended**: 2560 x 1600 pixels (portrait) or 1600 x 2560 pixels
- **Aspect Ratio**: 1:1.5 to 1:1.6 (portrait)
- **Maximum File Size**: 4MB
- **DPI**: 300 minimum
- **Color Mode**: RGB

#### Google Play Books
- **Format**: JPEG or PNG
- **Minimum Size**: 1600 x 2560 pixels
- **Aspect Ratio**: 1:1.6 (exact)
- **Maximum File Size**: 10MB
- **DPI**: 300
- **Color Mode**: RGB
- **Note**: No borders, text must be readable at thumbnail size

#### Amazon Kindle (KDP)
- **Format**: JPEG or TIFF (JPEG recommended)
- **Ideal Size**: 2560 x 1600 pixels (portrait)
- **Minimum**: 1000 pixels on longest side
- **Aspect Ratio**: 1:1.6 (8:5 ratio)
- **Maximum File Size**: 50MB
- **DPI**: 300
- **Color Mode**: RGB or sRGB
- **Note**: Cover must be professional quality

---

## Recommended Cover Dimensions
**Master File**: **2700 x 4050 pixels** (1:1.5 ratio)
This can be scaled down for all platforms without losing quality.

## Design Concept Recommendations

### **Visual Elements**

#### Theme: Geological Transformation
```
Layer 1 (Background): Desert landscape transitioning to digital grid
Layer 2 (Middle): Cracked earth with light emerging
Layer 3 (Foreground): Typography with Arabic and English
```

#### Color Palette
**Primary Colors:**
- **Desert Gold**: #C9A962 (warmth, heritage, Sudan)
- **Deep Blue**: #1B365D (stability, Saudi Arabia, technology)
- **Crack White**: #F5F5F5 (light through cracks, hope)

**Accent Colors:**
- **Nile Blue**: #4A90A4 (connection to origins)
- **Tech Silver**: #8B9DC3 (innovation, AI)
- **Earth Brown**: #6B4423 (grounding, roots)

#### Typography
**Title Font:**
- English: **Cinzel Bold** or **Playfair Display Bold** (elegant, authoritative)
- Arabic: **Cairo Bold** or **Tajawal Bold** (modern, readable)

**Subtitle Font:**
- English: **Montserrat Light** or **Open Sans Light**
- Arabic: **Tajawal Light** or **Amiri Regular**

**Author Name:**
- English & Arabic: Same as subtitle font, slightly smaller

## Design Elements to Include

### Must Have:
✅ Bilingual title (Arabic and English)
✅ Author name in both languages
✅ Visual reference to geological/transformation theme
✅ Professional, clean design
✅ Readable at thumbnail size (500px wide)

### Should Include:
✅ Subtitle or tagline
✅ Visual contrast between past and present
✅ Symbolic elements (crack, light, layers)
✅ Cultural authenticity

### Avoid:
❌ Cluttered design
❌ Too many fonts (max 2-3 font families)
❌ Stock photos that look generic
❌ Text smaller than 12pt at print size
❌ Low-contrast color combinations

## Brand Consistency

### BrainSAIT Logo Placement
- **Optional**: Small logo on spine or back cover
- **Not Recommended**: Logo on front cover (keep focus on book)
- **Use**: Professional variation with tagline
  - "Solutions that are Automated and Integrated with Technology"

### Author Branding
- **Professional Photo**: High-resolution headshot
- **Consistent Bio**: Use same bio across all platforms
- **Social Media**: Match cover aesthetics

## Design Tools Recommendations

### Professional (Recommended):
- **Adobe InDesign** (Complete control)
- **Adobe Photoshop** (Image editing)
- **Canva Pro** (User-friendly, templates available)

### Budget-Friendly:
- **GIMP** (Free Photoshop alternative)
- **Inkscape** (Vector graphics)
- **Canva Free** (Limited but functional)

### For Arabic Typography:
- **Adobe Creative Suite** (Best Arabic support)
- **Affinity Publisher** (Good alternative)
- Ensure **RTL (Right-to-Left)** support

## Quality Checklist

Before uploading to any platform:

### Technical:
- [ ] Correct dimensions for platform
- [ ] 300 DPI minimum
- [ ] RGB color mode
- [ ] File size within limits
- [ ] JPEG quality 90%+
- [ ] No transparency issues

### Design:
- [ ] Title readable at thumbnail (500px)
- [ ] Arabic text properly kerned
- [ ] No spelling errors (both languages)
- [ ] Professional quality imagery
- [ ] Color contrast sufficient
- [ ] Brand consistency

### Legal:
- [ ] Copyright notice present
- [ ] No unauthorized images/fonts
- [ ] Author photo permission
- [ ] ISBN placement (if printed)
